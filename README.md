# Calorie-counter

This is SPA which allows calculate individual daily calorie allowance

### [Demonstration](https://calories-navy.vercel.app)

### Install

```bash
$ git clone https://github.com/Rema04Dev/calories

$ cd calories

$ npm ci

$ yarn dev
or
$ npm run dev
```
